## 2. 개요
<p>
        패스트 패션 산업은 환경 파괴를 초래하는 주요 요인 중 하나로, 의류 산업은 세계 온실 가스 배출량의 약 10%를 차지한다. 이를 해결하기 위해 의류 관리를 신중하게 할 수 있도록 케어 라벨에 표기된 세탁 기호의 의미를 알려주는 시스템을 제작하였다.

</p>
